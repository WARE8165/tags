TAGS


!DOCTYPE html

    !DOCTYPE html, é a tag principal para o seu navegador identificar a versão do seu html.


HTML

    HTML, essa tag é responsável pelo funcionamento de outras tags.


HEAD

    HEAD, com o uso dessa tag, você criará o seu cabeçalho.


TITTLE

    TITTLE, com essa tag você cria o título do seu site.


LINK

    LINK, se você é usuário da tag style, saiba que com essa tag você pode linkar o CSS nela e deixar tudo mais organizado para alterar o estilo de tudo que você faz


BODY

    BODY, o body é a tag que é responsável por criar o corpo do seu site, e tudo aquilo que pode ser implementado.


H1

    H1, bem, com essa tag você poderá alterar a fonte dos seus textos, inclusive essa tag vária de tamanho, ela vai do h1 até p h6, começa a maior sendo h1 e termina a menor sendo h6.
